\section{文档范围与模块划分}

\begin{itemize}[nosep]
  \item 8 类物理层信号；
  \item 17 类物理层控制信息；

  \item 通信域与直通链路 PIB 定义及散布资源枚举。

\section{功能定义}

6.2.1 建立物理层\textbf{分类体系}与\textbf{信息块打包规则}，使接收端能够：
\begin{enumerate}[nosep]
  \item \textbf{分类识别}：区分 RE 上承载的是信号、控制还是数据；
  \item \textbf{块边界判定}：按 PIB 类型（SAB/RAB/G-PCIB/P-PCIB/A-PDIB）确定块边界与解码入口；
  \item \textbf{域类型区分}：识别通信域（经 G 节点，含 G/T/附链路）与直通链路（设备直连）的资源组织差异；
  \item \textbf{链路前缀解析}：按 G/T/附/直通链路前缀理解同名信号/控制在不同链路上的发送方向；
  \item \textbf{索引下游条款}：将每类内容映射至 6.2.3/6.2.4/6.2.5 的具体实现子节。
\end{enumerate}

\section{在 SLB 通信流程中的角色}

6.2.1 处于 PHY \textbf{内容组织最顶层}：在``帧结构已定''（6.2.2）与``具体波形生成''（6.2.3\textasciitilde 6.2.5）之间，回答``空口上传什么、放在哪个 PIB/链路/流程阶段''。

\textbf{通信域接收故事线（与 8 类信号出现顺序对应）：}
\begin{enumerate}[nosep]
  \item \textbf{搜网}：盲检 FTS/STS（同步信号）$\rightarrow$ 读 SAB 同步信息；
  \item \textbf{获知配置}：读广播信息（控制，散布资源）；
  \item \textbf{接入}：RAB 上 STS/FTS + 接入信息；
  \item \textbf{跟调度}：G-PCIB + G-DMRS-C $\rightarrow$ 解析 DSCI/预配置等；
  \item \textbf{传数据}：调度 RE 上的 DMRS-D + 第一/二类数据 $\pm$ 相位调整信号；
  \item \textbf{闭环}：CSI-RS/SRS 测量 $\rightarrow$ CQI；T-DMRS-C 伴随 TCI 反馈；
  \item \textbf{定位}：按资源指示在指定符号发/收位置测量信号；
  \item \textbf{空闲 RE}：未调度则不解析（$a_{k,l}{=}0$）。
\end{enumerate}

\textbf{直通链路简化故事线：}SAB $\rightarrow$ P-PCIB（直通 DSCI）$\rightarrow$ P-DMRS-C + DMRS-D + 数据。

\textbf{流程上下文}：T 节点尚未建立连接时，通过 6.2.7 3D-FISA \textbf{盲检 STS/FTS} 锁定 G 节点 COT；G 节点多域同步（6.2.8）在步骤二捕获邻域 FTS。同步信号\textbf{不承载高层比特}，但 SAB 内 \#3\textasciitilde\#4 紧跟同步信息（控制）。

\textbf{关联内容}：6.2.4.1 同步信息、6.2.4.2 SAB 周期；6.2.3.1 序列公式。

\subsubsection{2. G 链路公共解调参考信号（G-DMRS-C）}

\textbf{功能定位}：为\textbf{G 链路物理层控制信息}（主要是 G-PCIB 内 GCI）提供公共信道估计，接收端先估计 $H(k,l)$ 再 QPSK 解调控制比特。

\textbf{空口所在位置}
\begin{itemize}[nosep]
  \item \textbf{含 SAB 的 TTI}：该 TTI 第一个无线帧的 \textbf{\#6 符号}，全带宽 161 子载波（$k{\neq}80$）；
  \item \textbf{不含 SAB 的 TTI}：\textbf{\#0 符号}；
  \item \textbf{约束}：\textbf{不包含 SAB 的基础载波上不映射} G-DMRS-C。
\end{itemize}

\textbf{流程上下文}：终端完成 SAB 同步后，在每个 TTI 的 G-PCIB 盲检之前/并行接收 G-DMRS-C；功率与 G 链路控制信息子载波平均功率相同（6.5.4）。

\textbf{关联内容}：G-PCIB（6.2.4.7）、DSCI/预配置/HARQ 指示等 GCI（6.2.4.6）；与 P-DMRS-C（直通 \#6）结构类似但链路不同。

\subsubsection{3. T 链路控制信息解调参考信号（T-DMRS-C）}

\textbf{功能定位}：为\textbf{T 链路控制信息 TCI}（单级/两级 HARQ、CQI、调度请求）提供上行解调参考；G 节点侧用于估计 T$\rightarrow$G 反馈信道。

\textbf{空口所在位置}
\begin{itemize}[nosep]
  \item 映射在 TCI 发送符号的\textbf{第一个符号}；
  \item 频域资源（基础载波 + 梳齿集合）与对应 TCI \textbf{完全相同}；
  \item 基准 TTI 由触发 TCI 的 DSCI 或预配置激活去激活信息所在 TTI 确定（SR 除外）。
\end{itemize}

\textbf{流程上下文}：下行数据调度（DSCI）$\rightarrow$ T 在指示的基础载波/符号上发 ACK/NACK 或 CQI $\rightarrow$ 首符号 T-DMRS-C + 后续符号 TCI 比特/符号。功率按梳齿占有效子载波比例做 power boosting。

\textbf{关联内容}：6.2.4.10 TCI 全集；DSCI 中``TCI 所在基础载波序号''字段。

\subsubsection{4. 数据信息解调参考信号（DMRS-D）}

\textbf{功能定位}：为第一/二类物理层数据的均衡与解调提供参考；支持多端口、连续或梳齿子载波、多符号 DMRS 组。

\textbf{空口所在位置}
\begin{itemize}[nosep]
  \item 映射在\textbf{被调度传输数据的 RE 网格}内，由 DSCI/预配置指示时频资源；
  \item 每组 DMRS 占连续 $N = N_{\mathrm{port}}/N_{\mathrm{comb}}$ 个符号（$N_{\mathrm{comb}}{\in}\{1,2\}$）；
  \item 跨含/不含 SAB 载波调度时，时域起点以\textbf{含 SAB 载波}为基准（6.2.3.6）。
\end{itemize}

\textbf{流程上下文}：空口\textbf{用户面核心}——控制面完成调度后，数据 RE 与 DMRS-D 同 TTI 映射；附链路 A-PDIB 从 \#4 起周期映射 DMRS-D（6.2.5.4）。

\textbf{关联内容}：6.2.5 第一/二类数据；6.2.4.6.1 DSCI 资源字段；A-PDIB 布局。

\subsubsection{5. 数据信息相位调整信号（PAS）}

\textbf{功能定位}：在\textbf{跨多个数据符号}传输时，在特定 RE 上发送已知参考，接收端跟踪相位变化（频偏残留、预编码相位旋转等），保证数据符号相位一致。

\textbf{空口所在位置}
\begin{itemize}[nosep]
  \item 映射在\textbf{数据帧第一个符号}上由高层配置的两个子载波 $k_0, k_1$；
  \item 与同一数据调度绑定，不独立占用 PIB。
\end{itemize}

\textbf{流程上下文}：高阶调制、多符号数据块或 MIMO 预编码场景下，在 DMRS-D 间隔较大的符号之间辅助相位跟踪；属于\textbf{散布资源}，非 SAB/G-PCIB 固定结构。

\textbf{关联内容}：6.2.3.8 映射公式；6.2.5 数据 RE 布局。

\subsubsection{6. 信道探测信号（SRS）}

\textbf{功能定位}：T 节点\textbf{主动上行探测}，G 节点测量 T$\rightarrow$G 信道，用于上行调度、波束/链路适配。

\textbf{空口所在位置}
\begin{itemize}[nosep]
  \item \textbf{T 链路符号}上按 \texttt{srs-Set-Conf} 配置的 $(k,l)$ 映射 $r_{n,l}(k)$；
  \item 由\textbf{预配置调度激活去激活信息}激活/去激活，非常驻 SAB 结构。
\end{itemize}

\textbf{流程上下文}：接入后链路优化阶段；G 侧收集 SRS $\rightarrow$ 调整 T 链路 MCS/资源。与 CSI-RS 形成\textbf{上下行互补}（SRS 测上行，CSI-RS 供下行 CQI）。

\textbf{关联内容}：6.2.4.6.2 预配置激活；6.2.3.2。

\subsubsection{7. 信道状态信息参考信号（CSI-RS）}

\textbf{功能定位}：G 节点发送\textbf{下行参考}，T 节点测量信道质量并生成 CQI 反馈，供 G 调度器选择 MCS 与资源。

\textbf{空口所在位置}
\begin{itemize}[nosep]
  \item \textbf{G 链路}符号；工作子载波组由预配置频域指示，梳齿由 \texttt{csi-RS-Set-Conf} 配置；
  \item \textbf{含 SAB 载波}：起始符号 = G-PCIB 之后第一个符号；
  \item \textbf{不含 SAB 载波}：若 \#0 为 STS 则 STS 后第一个符号，否则 TTI 首符号。
\end{itemize}

\textbf{流程上下文}：闭环链路自适应——CSI-RS 测量 $\rightarrow$ TCI 中 CQI（6.2.4.10.3）$\rightarrow$ G 调整 DSCI 中 MCS 索引。

\textbf{关联内容}：6.2.4.10.3 CQI；6.2.5 表 8/9/10 MCS。

\subsubsection{8. 位置测量信号（PMS）}

\textbf{功能定位}：支持节点间\textbf{测距与定位}（如 RTT、AoA）；采用 ZC 型专用序列，可与安全机制（6.2.3.9 安全 PMS）结合。

\textbf{空口所在位置}
\begin{itemize}[nosep]
  \item \textbf{非固定 PIB 结构}：由 G 链路/T 链路\textbf{位置测量资源指示信息}（6.2.4.6.6/6.7）在当次 TTI 内指定起始符号、端口数、重复次数、波束数等；
  \item 可配置 G 发/T 收、T 发/G 收、双边两信号、三信号等测量图案。
\end{itemize}

\textbf{流程上下文}：业务数据之外\textbf{按需插入}；测量组建立后，锚点/标签按资源指示在指定符号发 PMS，对端接收并计算时延/角度。与 6.6 位置测量流程衔接。

\textbf{关联内容}：6.2.4.6.6 四格式资源指示；6.2.4.6.7 T 链路位置指示。

\section{约束与实现要点}
%----------------------------------------------------------------------

\begin{enumerate}[nosep]
  \item \textbf{【先同步后解调】}：任何控制/数据解析均依赖 FTS/STS 建立的定时与 SAB 中的系统参数。
  \item \textbf{【信号语义】}：8 类信号均为接收机辅助波形，不承载用户比特；用户载荷仅在 6.2.5 两类数据中。
  \item \textbf{【链路前缀】}：同名信号/控制按 G/T/附/直通前缀区分发送方向，实现时不得混用链路类型。
  \item \textbf{【G-DMRS-C 载波约束】}：无 SAB 的基础载波不发送 G-DMRS-C，盲检 G-PCIB 时需结合 SAB 存在性。
  \item \textbf{【DMRS 与数据绑定】}：DMRS-D 资源随 DSCI/预配置变化，不得按固定帧格式硬编码。
  \item \textbf{【域类型隔离】}：通信域 PIB（RAB/G-PCIB/A-PDIB）不适用于直通链路。
  \item \textbf{【空闲 RE】}：未分配 RE 对应 $a_{k,l}{=}0$，接收端不得当作有效载荷。
  \item \textbf{【中心子载波】}：$a_{80,l}\equiv 0$，任何内容均不映射至 \#80 子载波。
  \item \textbf{【6.2.1 与 6.2.3 映射】}：6.2.1 为逻辑分类，6.2.3.1\textasciitilde 10 为波形级定义（含感知扩展）。
  \item \textbf{【PIB 与散布资源】}：广播、DMRS-D、SRS、CSI-RS、PAS、PMS 等不进四块 PIB 整包，须单独解析映射规则。
  \item \textbf{【无状态机】}：实现用显式条件/标志/事件判断推进（域类型、SAB 存在性、调度指示），不维护 FSM 跳转表，避免异常路径锁死。
\end{enumerate}
